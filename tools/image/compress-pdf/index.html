FAQ

Each page is rendered as an image at your chosen resolution, then re-assembled into a new PDF with JPEG compression. This drastically reduces file size for image-heavy or scanned PDFs.
Setting the resolution scale above 100% renders pages at a higher DPI, producing sharper output. This is useful for low-resolution scanned documents that need better print quality.
